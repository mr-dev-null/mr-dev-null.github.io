Projects & Tools

Open-source tools, scripts, and utilities built for malware analysis, reverse engineering, and security automation. All MIT licensed unless noted.

Active

  • malscope

    Automated static analysis pipeline for Windows PE and Linux ELF malware samples. Extracts imports, strings, and entropy sections; detects packers; and generates YARA rules automatically. Clean CLI, structured JSON output.

    Go·YARA ·PE / ELF
    github →
  • ptrace-tracer

    Linux syscall interceptor built on ptrace(2) — a minimal strace reimplementation in Go. Reads registers, decodes syscall arguments with types, handles forked children, and outputs structured JSON traces. Useful for malware sandbox analysis.

    Go·Linux ·ptrace
    github →
  • proxyhunter

    High-speed concurrent proxy scraper and validator. Scrapes multiple sources in parallel, validates with configurable timeout and judge URLs, exports to TXT / CSV / JSON. Atomic round-robin rotation built in.

    Go·HTTP ·Concurrency
    github →
  • re-toolkit

    A collection of Ghidra and IDA Python scripts for automating repetitive reverse engineering tasks: renaming functions by pattern, recovering Go symbols, bulk decompiling to files, and detecting obfuscation techniques.

    Python·Ghidra ·IDA
    github →
